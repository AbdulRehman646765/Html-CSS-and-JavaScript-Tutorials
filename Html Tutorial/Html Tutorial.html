1. HTML HOME

Definition: HTML (Hypertext Markup Language) aik basic language hai jo web pages aur applications ka structure banane ke liye use hoti hai. Yeh elements ka use karke text, images, links, aur forms ko web page par organize karne mein madad karta hai.

Syntax:

<!DOCTYPE html>
<html>
<head>
  <title>Home Page</title>
</head>
<body>
  <h1>Welcome to HTML</h1>
  <p>HTML ka basic structure yahaan dekhein.</p>
</body>
</html>

Example: Yeh simple HTML structure hai jo web page ka basic layout define karta hai.

11. HTML Abbreviations and Quotes

<abbr>

Definition: <abbr> tag abbreviation ya acronym ko define karne ke liye use hota hai. Is tag ke andar title attribute ka istemal kiya ja sakta hai, jo full form ya meaning ko specify karta hai.

Example:

<abbr title="Hypertext Markup Language">HTML</abbr>

Yahan, HTML abbreviation hai, jiska full form "Hypertext Markup Language" hai.

Output:

HTML

<address>

Definition: <address> tag contact information ko define karta hai jo document ke author ya owner ka hota hai. Ismein email addresses, phone numbers, ya physical addresses shamil ho sakte hain.

Example:

<address>123 Main St, Anytown, USA<br>Email: <a href="mailto:example@example.com">example@example.com</a></address>

Yahan, address tag mein ek physical address aur email address diya gaya hai.

Output:

123 Main St, Anytown, USA
Email: example@example.com

<bdo>

Definition: <bdo> tag text direction ko define karta hai. Is tag ka istemal jab text ka direction reverse karna ho, tab kiya jata hai. Ismein dir attribute ka istemal hota hai, jismein ltr (left to right) ya rtl (right to left) specify kiya ja sakta hai.

Example:

<bdo dir="rtl">Yeh text right to left hai.</bdo>

Output:

Yeh text left to right hai.
Yeh text right to left hai.

<blockquote>

Definition: <blockquote> tag ek section ko define karta hai jo dusre source se quoted hota hai. Yeh generally long quotations ke liye use hota hai aur ismein cite attribute specify kiya ja sakta hai, jo source ki URL provide karta hai.

Example:

<blockquote cite="https://example.com">Yeh ek quoted text hai.</blockquote>

Output:

Here is a quote from WWF's website:

For 60 years, WWF has worked to help people and nature thrive. As the world's leading conservation organization, WWF works in nearly 100 countries. At every level, we collaborate with people around the world to develop and deliver innovative solutions that protect communities, wildlife, and the places in which they live.

<cite>

Definition: <cite> tag kisi work ka title define karta hai, jaise ke book, poem, ya film. Iska use citations ko represent karne ke liye kiya jata hai.

Example:

<cite>Zarb e Kaleem</cite> by Dr Allama Iqbal.

Yahan, Zarb e Kaleem ek book ka title hai.

Output:

Zarb e Kaleem by Dr Allama Iqbal.

<q>

Definition: <q> tag inline quotations ko define karta hai. Yeh short quotations ke liye use hota hai aur automatically quotation marks ke sath display hota hai.

Example:

<q>This is an inline quotation.</q>

Yahan, inline quotation ko q tag ke zariye represent kiya gaya hai.

Output:

WWF's goal is to: Build a future where people live in harmony with nature.

14. HTML Styles - CSS

1. CSS (Cascading Style Sheets)

Definition: CSS ek stylesheet language hai jo HTML documents ke presentation ko control karti hai. Iska istemal web pages ko visually appealing banane ke liye hota hai, jismein text ki styling, layout, colors, aur spacing shamil hoti hain. CSS ka design aur layout ko alag karne ka tareeqa hai, jis se code ko maintain karna asaan hota hai.

2. CSS Syntax

Definition: CSS ka syntax selectors aur declarations se milkar banta hai. Selector wo element hai jis par style apply hota hai, aur declaration wo properties aur values hain jo apply hoti hain.

Example:

h1 {
  color: blue;
  font-size: 24px;
}

Yahan, h1 selector hai aur us par color aur font-size properties apply ki gayi hain.

3. Inline CSS

Definition: Inline CSS ko HTML elements ke andar style attribute ke zariye directly define kiya jata hai. Yeh method sirf specific element par style apply karta hai.

Example:

<p style="color:red;">Yeh red color ka paragraph hai.</p>

4. Internal CSS

Definition: Internal CSS ko <style> tag ke andar define kiya jata hai, jo <head> section mein hota hai. Iska istemal ek hi document ke liye styles define karne ke liye kiya jata hai.

Example:

<style>
  p {
    font-family: Arial;
    background-color: yellow;
  }
</style>

5. External CSS

Definition: External CSS ko alag .css file ke roop mein create kiya jata hai aur HTML document ke <head> section mein <link> tag ke zariye link kiya jata hai. Is method ka istemal styling ko alag rakhne ke liye hota hai, jisse multiple pages par wohi styles apply kiye ja sakte hain.

Example:

<link rel="stylesheet" href="styles.css">

Is example mein, styles.css file ko HTML document se link kiya gaya hai.

6. CSS Properties

Definition: CSS mein kai properties hain jo elements ke styling ko define karti hain, jaise color, background-color, font-size, margin, aur padding. In properties ka istemal layout aur design ko enhance karne ke liye kiya jata hai.

Example:

div {
  background-color: lightgrey;
  margin: 20px;
  padding: 10px;
}

19. HTML Tables

Definition: Tables ko <table> tag ke zariye create kiya jata hai, jismein rows aur columns hote hain.

Table Tags Explained

  • <table> : Table ko define karta hai.
  • <tr> : Table row ko define karta hai.
  • <th> : Table header cell ko define karta hai, jo generally bold aur centered hota hai.
  • <td> : Table data cell ko define karta hai, jo regular content ko represent karta hai.

Example of an HTML Table

Example:

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Abdul Rehman</td>
    <td>24</td>
  </tr>
  <tr>
    <td>Bilal Hassan</td>
    <td>27</td>
  </tr>
</table>

Output:

Name Age
Abdul Rehman 24
Bilal Hassan 27

30. HTML Responsive Web Design

Responsive web design ka matlab hai ke website har device par acche se dikhe. Ismein media queries, fluid grids, aur responsive images ka istemal hota hai.

Media Queries: CSS ka feature hai jo screen size ke hisaab se styles apply karta hai.
Example: @media (max-width: 600px) { body { background-color: lightblue; } }

Fluid Grids: Layout ko percentage mein define karte hain, taake woh screen size ke sath adjust ho sake.
Example: .container { display: flex; flex-wrap: wrap; }

Responsive Images: Aise images jo har device par acche se load hote hain.
Example: <img src="small.jpg" srcset="medium.jpg 600w, large.jpg 1200w" alt="Image Description">
