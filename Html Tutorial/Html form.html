HTML Forms

HTML Forms

Definition: HTML Forms web pages par user input collect karne ke liye banaye gaye hain. Yeh mostly login, registration aur feedback forms mein use hote hain, jahan user se data liya ja sakta hai aur server par bheja ja sakta hai.

Example:

<form action="/submit" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <label for="email">Email:</label>
  <input type="email" id="email" name="email">
  <input type="submit" value="Submit">
</form>
