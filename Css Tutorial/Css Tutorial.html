2. CSS Introduction

Example:

  color: blue;
  font-size: 24px;
}

Output:

Applies blue color and font size to <h1> elements.

CSS Pseudo-class

Definition: CSS pseudo-classes are used to define the special state of an element. They allow you to apply styles based on the element's state or its relationship to other elements in the document. Pseudo-classes are defined by a single colon (:) before the class name.

Common Pseudo-classes:

  • :hover: Applies styles when the user hovers over an element with the mouse.
  • :active: Applies styles when an element is being activated by the user (e.g., when a button is pressed).
  • :focus: Applies styles when an element, such as an input field, receives focus.
  • :visited: Applies styles to links that have already been visited by the user.
  • :nth-child(n): Applies styles to elements based on their order among their siblings. "n" can be a number, keyword, or formula.
  • :first-child: Targets the first child element of a parent.
  • :last-child: Targets the last child element of a parent.

Example:

a:hover {
  color: red;
}

Output:

Changes the color of a link to red when hovered over.

Example:

input:focus {
  border: 2px solid blue;
}

Output:

Changes the border color of an input field to blue when it is focused.

Link of extra information for this topic:

CSS Pseudo-class

CSS Pseudo-element

Definition: CSS pseudo-elements allow you to style specific parts of an element. They enable you to apply styles to elements without adding extra markup in the HTML. Pseudo-elements are defined by a double colon (::) syntax, distinguishing them from pseudo-classes, which use a single colon (:).

Common Pseudo-elements:

  • ::before: Inserts content before the content of the selected element.
  • ::after: Inserts content after the content of the selected element.
  • ::first-line: Targets the first line of a block-level element and allows you to style it differently.
  • ::first-letter: Targets the first letter of a block-level element, allowing for unique styling.
  • ::selection: Styles the portion of a document that has been highlighted by the user.
  • ::placeholder: Styles the placeholder text in an input element.

Example:

p::first-line {
  font-weight: bold;
  font-size: 1.2em;
}

Output:

Makes the first line of a paragraph bold and slightly larger.

Example:

h1::after {
  content: " - Welcome!";
  color: gray;
}

Output:

Adds " - Welcome!" in gray color after every <h1> element.

CSS Math Functions

Definition: CSS Math Functions aapko CSS mein calculations karne ki ijaazat dete hain. Yeh functions web development mein flexibility aur dynamic layouts banane ke liye bohot faydemand hain. Yeh kuch popular functions hain:

1. calc(): Is function ka istemal aapko do ya zyada values ko combine karne ki ijaazat deta hai. Yeh different units ke saath bhi kaam kar sakta hai, jaise pixels (px), percentages (%), ems, etc.
Example: width: calc(100% - 50px); Yeh width ko parent element ki 100% width se 50 pixels kam karke set karta hai.

2. min(): Is function ka istemal kisi do values mein se choti value ko select karne ke liye hota hai. Yeh responsive design mein madadgar hota hai, jahan aap chahte hain ke element ki size kabhi bhi exceed na kare.
Example: width: min(50%, 300px); Yeh width ko 50% ya 300 pixels mein se choti value set karega.

3. max(): Is function ka istemal kisi do values mein se badi value ko select karne ke liye hota hai. Yeh aapko ensure karta hai ke element ki size kabhi bhi kam na ho.
Example: width: max(200px, 20%); Yeh width ko 200 pixels ya 20% mein se badi value set karega.

4. clamp(): Is function ka istemal kisi value ko minimum aur maximum limits ke beech constrain karne ke liye hota hai. Yeh function responsive design mein madadgar hai.
Example: font-size: clamp(16px, 5vw, 24px); Yeh font-size ko 16 pixels se 24 pixels ke beech rakhta hai, jo viewport width ke hisaab se adjust hota hai.

CSS Math Functions ki wajah se aap web design mein zyada flexibility hasil karte hain, jisse aapki website har device par behtar tarah se display hoti hai.
