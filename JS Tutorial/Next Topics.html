46. JS Loop For

Definition: JavaScript for loop ko repeated execution ke liye use karte hain jab tak ek condition true rehti hai.

Syntax:

for (initialization; condition; increment) {
// code to be executed
}

Example:

for (let i = 0; i < 5; i++) {
console.log(i);
}

Output:

0
1
2
3
4

47. JS Loop For In

59. JS Bitwise

Definition: JavaScript Bitwise operators integers ke bits par operations perform karte hain.

Syntax:

a & b; // Bitwise AND
a | b; // Bitwise OR
a ^ b; // Bitwise XOR
~a; // Bitwise NOT

Example:

let a = 5; // 0101 in binary
let b = 3; // 0011 in binary
console.log(a & b); // Output: 1 (0001)
console.log(a | b); // Output: 7 (0111)
console.log(a ^ b); // Output: 6 (0110)
console.log(~a); // Output: -6 (inverts bits)

Output:

1
7
6
-6
