Syntax:

script tag ke andar JavaScript likha jata hai: <script>

</script>

Example:

</script>

45. JS Switch

Definition: JavaScript Switch statement multiple conditions ka check karne ke liye use hota hai, jahan aap alag-alag cases specify kar sakte hain.

Syntax:

switch (expression) {
case value1:
// code to be executed if expression == value1
break;
case value2:
// code to be executed if expression == value2
break;
default:
// code to be executed if expression doesn't match any cases
}

Example:

let fruit = 'apple';
switch (fruit) {
case 'banana':
console.log('Banana selected');
break;
case 'apple':
console.log('Apple selected');
break;
default:
console.log('No fruit selected');
}

Output:

Apple selected
