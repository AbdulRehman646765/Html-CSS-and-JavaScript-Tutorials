77. JS Callbacks

Definition: Callback function woh function hota hai jo dusre function ko argument ke tor par diya jata hai aur woh function ki execution ke baad call hota hai.

Syntax:

function myFunction(callback) {
// Function code
callback();
}

Example:

function greet() {
console.log("Hello!");
}
myFunction(greet);

Output:

Hello!

78. JS Asynchronous

Definition: Asynchronous programming mein code aisa likha jata hai jo dusre code ki execution ke intezar ke baghair chalta hai, jisse performance improve hoti hai.

Syntax:

setTimeout(function() {
// Code to execute after delay
}, 1000);

Example:

console.log("Start");
setTimeout(function() {
console.log("Async operation");
}, 2000);
console.log("End");

Output:

Start
End
Async operation

79. JS Promises

Definition: Promise ek object hai jo future mein kisi value ko represent karta hai, jo abhi available nahi hai, lekin aage chal kar available ho sakti hai.

Syntax:

const myPromise = new Promise((resolve, reject) => {
// Asynchronous operation
if (/* operation successful */) {
resolve("Success!");
} else {
reject("Failure!");
}
});

Example:

myPromise.then(result => {
console.log(result);
}).catch(error => {
console.log(error);
});

Output:

Success!

80. JS Async/Await

Definition: Async/Await JavaScript mein asynchronous programming ko likhne ka aik asaan tareeqa hai jo promises ko use karte hue asynchronous code ko sequentially execute karta hai.

Syntax:

async function myFunction() {
const result = await myPromise;
console.log(result);
}

Example:

async function fetchData() {
const response = await fetch("https://api.example.com/data");
const data = await response.json();
console.log(data);
}

Output:

Data fetched from API
